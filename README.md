# Django Tree Menu App - DogDating

Приложение для знакомства собак с реализацией древовидного меню через template tags.

## Особенности

✅ Древовидное меню через template tag `{% draw_menu 'menu_name' %}`
✅ Развертывание активных пунктов и их родителей
✅ Хранение в БД с редактированием через админку Django
✅ Определение активного пункта по текущему URL
✅ Поддержка нескольких меню на странице
✅ Поддержка явных URL и named URLs
✅ **Только 1 запрос к БД на отрисовку каждого меню**

## Установка

1. Клонируйте репозиторий
2. Создайте виртуальное окружение
3. Установите зависимости: `pip install -r requirements.txt`
4. Выполните миграции: `python manage.py migrate`
5. Создайте суперпользователя: `python manage.py createsuperuser`
6. Загрузите тестовые меню: `python manage.py setup_menus`
7. Запустите сервер: `python manage.py runserver`
